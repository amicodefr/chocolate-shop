# Sweet Dreams 🍭

**Интернет-магазин вкусняшек** — современное React-приложение с удобной корзиной и красивым интерфейсом.



## ✨ Возможности

- 🌟 Красивый и удобный дизайн
- 🛒 Полноценная корзина с сохранением в `localStorage`
- 📱 Адаптивная вёрстка (мобильная версия)
- 🔍 Поиск по товарам (можно включить)
- ⚡ Быстрая загрузка товаров через API
- 🛍️ Красивые карточки товаров
- 📄 Несколько страниц (Главная, Меню, Корзина, О нас)

## 🛠 Технологии

- **React** + Vite
- **Redux Toolkit** 
- **React Router** v6
- **Axios** 
- CSS Modules

## 🚀 Как запустить проект локально

```bash
# 1. Клонируй репозиторий
git clone https://github.com/твой-username/sweet-dreams.git

# 2. Перейди в папку проекта
cd sweet-dreams

# 3. Установи зависимости
npm install

# 4. Запусти проект
npm run dev
